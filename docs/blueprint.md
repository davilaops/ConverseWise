# **App Name**: ConversaWise

## Core Features:

- Content Transcription: Transcribes audio and video content from various sources (YouTube, Zoom, Meet, Telegram, uploaded files) into text with high accuracy using the OpenAI Whisper API, supporting multiple languages.
- Intelligent Summarization: Generates structured summaries of transcribed content using GPT-4-Turbo, including main points, decisions, speaker identification, insights, and next steps. Offers adjustable summary types (quick, complete, executive).
- Export and Sharing: Exports summaries in PDF and DOCX formats, and enables direct sharing via WhatsApp and email.
- Multilingual Support: Provides full support for Portuguese and English, including automatic language detection of audio content.
- User History and Storage: Stores summaries per user for access in a personal dashboard, allowing users to review past transcriptions and summaries.
- Account and Subscription Management: Handles user authentication via Google OAuth and email, manages subscriptions through Stripe with a free trial and paid monthly/annual plans.
- Highlight Clipping Tool: Allows users to select and clip important sections of the transcribed text to generate short text or video highlights.

## Style Guidelines:

- Primary color: Light sky blue (#A878F5) to convey trust and clarity.
- Background color: Very light grey (#F0F0F0) to maintain a minimalist and clean aesthetic.
- Accent color: Soft lavender (#E6E6FA) to provide subtle highlights and call to action elements.
- Body and headline font: 'Inter', a sans-serif, to offer a modern and neutral readability.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean and consistent line icons throughout the interface to represent different actions and content types.
- Maintain a minimalist and modern SaaS style with clear sections for upload/link input, summary display, and export/sharing options.
